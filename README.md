# Exoplanets

Web game · Board game promotion

## About the project

Exoplanets is a browser-based game created to promote my own board game.
Players can create and customize their own planets and progress by
earning rewards through in-app activity.

The project combines gameplay with a simple gamification system and full
user account functionality.

## Features

-   Planet creation and customization system\
-   User registration and login\
-   Reward and progression system (gamification layer)\
-   Persistent user data storage\
-   Browser-based gameplay

## Tech Stack

-   HTML\
-   CSS\
-   JavaScript\
-   PHP\
-   MySQL

## Goal

The goal of the project is to engage users through interactive gameplay
while promoting a physical board game in a digital environment
